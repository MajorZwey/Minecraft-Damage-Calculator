# Usage
1. [Download](https://github.com/MajorZwey/Minecraft-Damage-Calculator/releases/download/alpha_0.22/Minecraft-Damage-Calculator_alpha-0.22.jar) the .jar file.
2. Execute the jar file like this: jave - jar Minecraft-Damage-Calculator_x.xx.
3. Enter your values into the prompts and your damage will be calculated.
